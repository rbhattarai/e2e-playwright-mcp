# PROMPT Examples

Using the Playwright MCP tool, open playwright.dev, click "Get Started" and Navigate to "API Testing" can capture the example test case and generate a Playwright TypeScript test under e2e/api-test.spec.ts. 

Do not use the Github example as mentioned in API Testing page but make REST call to: https://jsonplaceholder.typicode.com and create tests to cover POST, GET, PUT and DELTE methods.
